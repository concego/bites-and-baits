Como jogar

📱 Celular com acelerômetro

  • Lançar: incline o celular para frente
  • Puxar: incline o celular para trás
  • Fisgar: sacuda o celular quando ouvir o aviso

  • Lançar: seta para cima ↑
  • Puxar: seta para baixo ↓
